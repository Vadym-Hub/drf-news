## Or get started with Docker

#### 1) Clone the git repository

    https://github.com/Vadym-Hub/drf-news

#### 2) Create an image

    docker-compose build .

##### 3) Start the container

    docker-compose up
    
##### 4) Go to the address

    http://127.0.0.1:8000/api/v1/swagger/
